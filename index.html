The Model 100 Archive is an open source (public domain) archive of BASIC software for the TRS-80 Model 100.

Contact us at archive@model100.in

Want to contribute to the archive? Submit a pull request to this repository or email us (archive@model100.in)

All the software in the Model 100 Archive is in the public domain, and free for commercial use.
